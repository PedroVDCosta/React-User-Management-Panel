# React-User-Management-Panel

A demonstration project built with React to showcase frontend development skills, REST API integration and data handling in a realistic application scenario.

---

## Features

* User listing from REST API
* Real-time search and filtering
* User detail view
* Error handling and loading states
* Responsive UI

---

## Technologies

* React
* JavaScript (ES6+)
* Fetch API
* HTML / CSS
* Component-based architecture

---

## Purpose

This project was developed to demonstrate practical skills in:

* Frontend development using React
* Integration with REST APIs
* Data manipulation and UI rendering
* Building interfaces for real-world applications

---

## How to Run

```bash
npm install
npm start
```
